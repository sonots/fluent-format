# fluent-format

A command line utility to format fluentd configuration beautifully

## Installation

Add this line to your application's Gemfile:

    gem 'fluent-format'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fluent-format

Run as

    $ fluent-format -c fluentd.conf
